🚀 Serverless Static Portfolio Website
Welcome to my personal portfolio repository! This project showcases a highly available, secure, and modern static website hosted entirely on AWS serverless architecture.

🌟 Architecture Overview
The website is built using HTML/CSS and deployed via a secure pipeline. It leverages AWS CloudFront as a Content Delivery Network (CDN) to serve content globally with low latency, while AWS S3 securely stores the static assets.

🛠️ What I Created and Why
Amazon S3 Bucket (muhammad-azam-portfolio-2026):

What it is: S3 is an object storage service where I uploaded the website's index.html file.

Why I built it: It serves as the industry-standard foundation for static website hosting, ensuring high security, scalability, and cost-effectiveness.

Amazon CloudFront (CDN Distribution):

What it is: AWS's Content Delivery Network (CDN) mapped directly to the S3 bucket.

Why I added it: It delivers the website globally with ultra-low fast loading speeds and enhances security by blocking direct public access to the S3 bucket via Origin Access Control (OAC).

Serverless Frontend (index.html & CSS):

What it is: A custom, responsive personal portfolio website designed using HTML5 and CSS3.

Why I built it: To professionally showcase my technical skills (AWS, DevOps, CI/CD) and cloud projects to potential employers.

Version Control & CI/CD Pipeline (GitHub & GitHub Actions):

What it is: Repository setup for secure code versioning and automated cloud deployments.

Why I set it up: To ensure that any future code updates on my local machine automatically sync and deploy to the live website without manual uploads.
